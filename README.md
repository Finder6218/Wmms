# 手表维修管理系统

《手表维修管理系统》是一款记录维修客户信息的软件,方便查询维修历史记录，易于管理客户。系统包含： 1、基本设置：单位信息 2、维修管理：维修登记、维修查询 



# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
